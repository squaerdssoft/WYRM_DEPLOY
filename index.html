WYRM AI'ya Giriş

Yapay Zeka Geleceğine Hoş Geldiniz

WYRM AI, geliştiriciler için tasarlanmış yeni nesil yapay zeka platformudur. Güçlü dil modelleri, görsel işleme yetenekleri ve akıllı fonksiyon çağırma özellikleri ile projelerinizi bir üst seviyeye taşıyın.

WYRM API'sine giriş rehberi. Bu bölümde temel kavramları, API'nin nasıl çalıştığını ve neler yapabileceğinizi detaylı şekilde öğreneceksiniz. Başlangıçtan ileri seviyeye kadar tüm bilgileri burada bulabilirsiniz.

🚀

WYRM Nedir?

WYRM (Web-scale Yet Responsive Model), gelişmiş yapay zeka yetenekleri sunan bulut tabanlı bir API platformudur. Stanford ve MIT'den gelen araştırmacılar tarafından geliştirilen WYRM, endüstri standardında performans ve güvenilirlik sunar.

Geliştiricilere en güçlü AI araçlarını basit ve erişilebilir bir şekilde sunmak. Karmaşık AI işlemlerini birkaç satır kodla gerçekleştirmenizi sağlıyoruz.

Quickstart Tutorial

API'ye Hızlı Başlangıç

WYRM API ile ilk isteğinizi yapmak için kapsamlı adım adım rehber. Bu tutorial, API'yi kullanmaya başlamanız için gereken tüm bilgileri içerir.

📋 Ön Koşullar

  • WYRM hesabı (ücretsiz kayıt olabilirsiniz)
  • Temel HTTP istekleri bilgisi
  • Terminal veya kod editörü erişimi

🔑 1. API Anahtarı Oluşturun

WYRM Dashboard'a giriş yapın ve API anahtarınızı oluşturun. Bu anahtar, tüm API isteklerinizde kimlik doğrulama için kullanılacak.

Adımlar:

  1. WYRM Dashboard'a giriş yapın
  2. Sol menüden "API Keys" sekmesine tıklayın
  3. "Create New Key" butonuna basın
  4. Anahtarınıza bir isim verin (örn: "My First Project")
  5. Oluşturulan anahtarı güvenli bir yere kaydedin
# Anahtarınızı environment variable olarak ayarlayın
export WYRM_API_KEY="wyrm-1234567890abcdef..."

# Veya .env dosyasında saklayın
echo "WYRM_API_KEY=wyrm-1234567890abcdef..." >> .env

🚀 2. İlk İsteğinizi Yapın

Basit bir metin tamamlama isteği ile API'yi test edin:

cURL ile:

curl -X POST "https://api.wyrm.ai/v1/chat/completions" \
  -H "Authorization: Bearer $WYRM_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "wyrm-1",
    "messages": [
      {"role": "user", "content": "Merhaba WYRM! Nasılsın?"}
    ],
    "max_tokens": 150,
    "temperature": 0.7
  }'

Python ile:

import requests
import os

url = "https://api.wyrm.ai/v1/chat/completions"
headers = {
  "Authorization": f"Bearer {os.getenv('WYRM_API_KEY')}",
  "Content-Type": "application/json"
}

data = {
  "model": "wyrm-1",
  "messages": [{"role": "user", "content": "Merhaba WYRM!"}],
  "max_tokens": 150
}

response = requests.post(url, headers=headers, json=data)
print(response.json())

JavaScript ile:

const response = await fetch('https://api.wyrm.ai/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${process.env.WYRM_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'wyrm-1',
    messages: [{role: 'user', content: 'Merhaba WYRM!'}],
    max_tokens: 150
  })
});

const data = await response.json();
console.log(data);

Tool Use (Function Calling)

WYRM'ın gelişmiş fonksiyon çağırma özelliği ile harici araçları güvenle entegre edin. Bu özellik, AI modelinizin gerçek dünya verileriyle etkileşim kurmasını ve karmaşık görevleri yerine getirmesini sağlar.

WYRM'ın fonksiyon çağırma özelliği, AI modelinin belirlediğiniz fonksiyonları akıllıca seçip çağırmasını sağlar. Model, kullanıcının isteğini analiz ederek hangi fonksiyonun gerekli olduğunu belirler ve uygun parametrelerle çağırır.

Basit bir hava durumu fonksiyonu tanımlayalım:

  "model": "wyrm-1",
  "messages": [
    {"role": "user", "content": "İstanbul'da hava nasıl?"}
  ],
  "tools": [{
    "type": "function",
    "function": {
      "name": "get_weather",
      "description": "Belirtilen şehir için güncel hava durumu bilgisi al",
      "parameters": {
        "type": "object",
        "properties": {
          "city": {
            "type": "string",
            "description": "Şehir adı"
          },
          "unit": {
            "type": "string",
            "enum": ["celsius", "fahrenheit"],
            "description": "Sıcaklık birimi"
          }
        },
        "required": ["city"]
      }
    }
  }]
}

2. Model Yanıtı:

          "name": "get_weather",
          "arguments": "{\"city\": \"İstanbul\", \"unit\": \"celsius\"}"

    }
  }]
}

Görsel İşleme ve OCR

Gelişmiş Görsel İşleme Teknolojisi

WYRM'ın güçlü görsel işleme yetenekleri ile resimlerden metin çıkarın, görselleri analiz edin ve karmaşık görsel verileri anlayın. OCR, nesne tespiti ve görsel soru-cevap özelliklerini keşfedin.

WYRM ile görsel analiz ve optik karakter tanıma (OCR) işlemleri yapabilirsiniz. Gelişmiş bilgisayar görüsü algoritmaları sayesinde resimlerden değerli bilgiler çıkarabilir ve görsel içeriği metne dönüştürebilirsiniz.

📸

Görsel İşleme Yetenekleri

WYRM, görselleri analiz edebilir, içeriğini açıklayabilir, metinleri okuyabilir ve karmaşık görsel sorulara yanıt verebilir. Makine öğrenmesi tabanlı algoritmalar ile yüksek doğruluk oranında sonuçlar elde edin.

🔍 Görsel Analiz

Resim içeriğini detaylı şekilde analiz eder, nesneleri tanır ve sahne kompozisyonunu açıklar.

📝 OCR Teknolojisi

Resimlerden metinleri yüksek doğrulukla çıkarır, el yazısı ve basılı metinleri tanır.

❓ Görsel Soru-Cevap

Resimler hakkında sorular sorun ve detaylı yanıtlar alın. Görsel içeriği anlayarak cevap verir.

📁

Desteklenen Formatlar

  • JPEG, JPG (yüksek kalite)
  • PNG (şeffaflık desteği)
  • WebP (modern format)
  • GIF (ilk kare analizi)
  • Base64 kodlu görseller
  • URL'den görsel yükleme
🎯

Kullanım Alanları

  • Belge dijitalleştirme
  • Fatura ve makbuz işleme
  • Kimlik belgesi doğrulama
  • Ürün katalog analizi
  • Medikal görüntü analizi
  • Güvenlik kamerası analizi
